# Pasta de Imagens

Adicione aqui as imagens do site.

## Imagens necessárias:

### Página Home
- `hero-image.jpg` - Imagem principal do hero section (recomendado: 800x600px)

### Geral
- `logo.png` - Logo da Vetorizando (opcional, caso queira substituir o texto)
- Imagens de serviços
- Fotos da equipe
- Fotos de clientes/cases

## Recomendações:

- **Formato:** JPG para fotos, PNG para logos/ícones
- **Otimização:** Comprima as imagens antes de fazer upload
- **Tamanho:** Máximo 500KB por imagem
- **Dimensões:** Proporção 16:9 para imagens de destaque
- **Alt text:** Sempre adicione descrições nas tags `<img>`

## Ferramentas de otimização:

- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac)
